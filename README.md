# CMRS — Rising Star Minerals Website

A marketing and informational website built for Rising Star Minerals, a mining company, covering company background, services, materials, and contact.

## Tech Stack
- Next.js (App Router) + TypeScript
- Next.js API routes for server-side email handling

## Pages
- **Home** — landing page
- **About** — company background
- **Services** — services offered
- **Materials** — materials/products information
- **Contact** — contact form with server-side email routing (no phone numbers exposed)

## Getting Started

\`\`\`bash
npm install
npm run dev
\`\`\`

Open [http://localhost:3000](http://localhost:3000) in your browser.
